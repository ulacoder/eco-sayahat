# EcoSayahat 🌿

> Платформа экологического туризма Казахстана

🌐 **[Открыть платформу](https://frontend-sandy-three-60.vercel.app/)**

## О проекте

**EcoSayahat** — веб-приложение для продвижения экологического туризма в Казахстане. Путешествуй по самым красивым регионам страны экологично, зарабатывай EcoCoins за заботу о природе и получай незабываемые впечатления!

### 🗺 Регионы Казахстана

Исследуй 5 уникальных природных зон:
- **Каспий** — побережье Каспийского моря
- **Бурабай** — национальный парк с горными озёрами
- **Алаколь** — солёное озеро с лечебными грязями
- **Балхаш** — уникальное полусолёное озеро
- **Кольсай** — горные озёра в Тянь-Шане

## ✨ Основные возможности

### 🧳 Для туристов

- **Виртуальные туры** — 360° панорамы достопримечательностей перед поездкой
- **Интерактивная карта** — маркеры локаций на OpenStreetMap
- **Эко-такси** — заказ электромобилей для передвижения по региону
- **Эко-отели** — бронирование экологичного жилья
- **Эко-задания** — выполняй задания (сортировка мусора, уборка территорий) и зарабатывай EcoCoins
- **AI-ассистент** — помощник по планированию маршрутов и ответам на вопросы
- **Мультиязычность** — русский, английский, казахский языки

### 🚕 Для водителей эко-такси

- **Карта зарядных станций** — быстро найди ближайшую точку зарядки электромобиля
- **Заказы в реальном времени** — получай и принимай заказы от туристов
- **Статистика поездок** — отслеживай свои поездки и заработок

### 👨‍💼 Для администраторов

- **Модерация отзывов** — управление отзывами пользователей
- **Аналитика платформы** — статистика по пользователям, заказам и активности
- **Управление контентом** — добавление новых локаций и достопримечательностей

## 💰 Система EcoCoins

**EcoCoins** — внутренняя валюта платформы для поощрения экологичного поведения.

### Как заработать:
- Выполнение эко-заданий (сортировка отходов, уборка территорий)
- Использование эко-такси вместо обычного транспорта
- Бронирование эко-отелей
- Участие в экологических акциях

### Где потратить:
- Скидки в партнёрских эко-отелях
- Бесплатные поездки на эко-такси
- Сувениры и мерч
- Доступ к премиум-контенту (эксклюзивные туры)

## 🎨 Дизайн

Зелёно-белая цветовая схема в стиле природы создаёт атмосферу свежести и экологичности. Интерфейс интуитивно понятен и работает на всех устройствах.

## 🛠 Технологии

### Frontend
- **React 19** — современная библиотека для UI
- **React Router v7** — навигация между страницами
- **Tailwind CSS** — utility-first CSS фреймворк
- **shadcn/ui** — готовые компоненты с Radix UI
- **React Leaflet** — интерактивные карты
- **i18next** — мультиязычность (KZ/RU/EN)
- **Axios** — HTTP клиент для API запросов

### Backend
- **Python Flask** — легковесный веб-фреймворк
- **JWT Authentication** — безопасная авторизация
- **Flask-CORS** — поддержка cross-origin запросов

### Сборка и инструменты
- **Create React App** — настройка проекта
- **Craco** — кастомизация конфигурации CRA
- **ESLint** — линтинг кода
- **PostCSS** — обработка CSS

## 🚀 Запуск проекта

### Frontend

```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

Приложение откроется на [http://localhost:3000](http://localhost:3000)

### Backend

```bash
cd backend
pip install -r requirements.txt
python server.py
```

API будет доступно на [http://localhost:5000](http://localhost:5000)

## 🌍 Особенности реализации

- **Адаптивный дизайн** — корректная работа на мобильных, планшетах и десктопах
